snmpsim使用教程

Posted on 2023-11-29 In 通信

1、服务器安装python和pip

类linux服务器一般默认安装python，可以输入python进行检验有无安装，如果安装了则会进入python交互界面，如未安装，需要自行安装python，一般安装python会自动安装pip。

2、安装snmpsim相关依赖

执行命令pip install pysnmp，一般会自动安装其它依赖的python包，如没有安装，则需要根据提示手动安装（提示缺少哪个包，则执行pip install xxx）。

3、下载snmpsim应用

在github上，通过https://github.com/etingof/snmpsim 下载snmpsim应用，通过sftp上传到相应的linux服务器，解压压缩包后，在文件夹内执行python setup.py install，安装snmpsim应用，执行完没有报错则是安装成功。

4、模拟snmp设备

执行 snmpsimd.py --data-dir=C:/Users/master/Desktop/snmp --args-from-file=C:/Users/master/Desktop/param.txt其中–data-dir表示数据文件夹里面包含public.snmprec文件，此文件一般在snmpsim文件夹下有默认的，可以使用，或者自己根据真实snmp设备自己制作，param.txt内容为需要模拟的snmp设备的ip和端口，格式如下图所示。

根据已有的mib制作public.snmprec

将需要模拟的mib文件放在当前用户的根目录下~/.pysnmp/mibs，并执行mib2dev.py会扫描此目录下的mib文件，执行指令为：mib2dev.py --output-file=./data/public.snmprec --mib-module=SNMPv2-MIB。生成后public.snmprec文件后执行步骤四即可。

根据真实设备制作public.snmprec

snmprec.py --agent-udpv4-endpoint=demo.snmplabs.com --output-file=./data/public.snmprec
